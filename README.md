# Blaz

## Building

- [Install CMake >= 3.12](https://cmake.org/download/)
- ```mkdir build```
- ```cd build```
- ```cmake ..```

## Features

| Graphics API  | Support |
| ------------- | ------------- |
| Opengl  | ✅ |
| Direct3D11 | ✅ |
| Direct3D12 | ❌ |
| Vulkan | ❌ |
